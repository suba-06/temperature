# Body Temperature Estimator Web App

## How to Use

1. Install dependencies:
   pip install flask opencv-python numpy

2. Run the backend:
   python flask_app.py

3. Open `index.html` in your browser.

4. Tap "Tap to Scan" and allow camera access if prompted.

The Python backend will read from your webcam, estimate the temperature, and return it to the web interface.
